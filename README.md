# WebAPP Merchant - Laravel MVC

WebAPP Merchant adalah aplikasi berbasis Laravel yang dikembangkan menggunakan konsep MVC (Model View Controller) untuk membantu proses pengelolaan data merchant secara digital. Aplikasi ini dilengkapi dengan fitur authentication, dashboard interaktif, upload gambar produk, visualisasi data menggunakan chart dan calendar, serta sistem role access antara Developer dan Merchant. User dengan role Developer memiliki akses untuk mengelola menu dan manajemen user, sedangkan user Merchant dapat mengelola kategori dan produk pada sistem katalog. Dengan adanya aplikasi ini, proses pengelolaan data produk menjadi lebih terstruktur, modern, dan mudah digunakan.

## ✨ Fitur

- Login & Register Authentication
- Dashboard Statistik
- Role Access Developer & Merchant
- CRUD Kategori
- CRUD Produk
- Upload Gambar Produk
- Diagram Chart Produk
- Calendar Dashboard
- Middleware Role
- Responsive Dashboard UI
